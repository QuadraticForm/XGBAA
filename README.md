This is an anti-aliasing method specifically designed for outlining.

Outlining here means outline rendered by offseting vertices along normal and rendering the back faces.

这是一个专门描边发明的抗锯齿方法。

这里的描边指以「法线外扩+渲染背面」方法渲染的描边。
